What is document object model (DOM)?

The Document Object Model (DOM) is a programming interface for web documents. It represents the structure of a webpage as a hierarchical tree of objects, where each part of the document (such as elements, attributes, and text) is represented as a node. The DOM allows programs, such as JavaScript, to access and manipulate the content, structure, and style of a webpage dynamically.

How do you select an element from the DOM?

To select an element from the DOM using JavaScript, you can use various built-in methods depending on what kind of element you're targeting. Here are some common ways to select elements:
1. getElementById()
2. getElementsByClassName()
3. getElementsByTagName()
4. querySelector()
5. querySelectorAll()

How do you manipulate an element's attributes and style using DOM?

In the DOM, you can easily manipulate an element's attributes and styles using JavaScript. Here's how you can achieve that:
1.getAttribute()
2.setAttribute()
3.removeAttribute()

What is the main difference between appendChild() and append()?

The main difference between appendChild() and append() is:
appendChild(): Only appends Node objects (like elements or text nodes), and only one node at a time. It returns the appended node.
append(): More flexible, allows appending multiple Nodes and text strings at once. It does not return anything.
append() is more versatile, while appendChild() is more limited but supported in all browsers.
